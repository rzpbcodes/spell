# 📚 Spell – Kids Spelling Practice App
A simple and interactive web app to help kids (Class 1–3) learn and practice English spellings through categorized lessons.

## ✨ Features
* 📖 Category-based lessons (Numbers, Colors, Animals, etc.)
* 🧠 Easy-to-learn word lists (basic → advanced)
* 🎯 Clean and simple UI for kids
